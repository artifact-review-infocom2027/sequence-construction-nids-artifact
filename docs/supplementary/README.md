# Supplementary Appendix

This folder provides reviewer-facing supplementary material for the anonymous
artifact. The files summarize the experimental pipeline, data handling,
model families, evaluation setup, and reproducibility scope using public
artifact paths.

The appendix material is written for artifact review and avoids editorial notes
from the manuscript source package.

## Contents

- [01_overview.md](01_overview.md): end-to-end pipeline overview.
- [02_datasets.md](02_datasets.md): dataset split and preprocessing summary.
- [03_schema_harmonization.md](03_schema_harmonization.md): schema and label
  harmonization notes.
- [04_feature_schema.md](04_feature_schema.md): public feature categories.
- [05_experimental_setup.md](05_experimental_setup.md): training and evaluation
  setup.
- [06_transformer_models.md](06_transformer_models.md): model-family summary.
- [07_design_guidelines.md](07_design_guidelines.md): reviewer-facing design
  guidelines supported by the evidence files.
- [08_additional_ablations.md](08_additional_ablations.md): additional ablation
  summaries and figure links.
- [09_reproducibility.md](09_reproducibility.md): public reproducibility scope
  and commands.
- [10_faq.md](10_faq.md): concise answers to common reviewer questions.
