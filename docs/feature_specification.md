# Feature Specification

The public feature specification describes the harmonized 47-field schema used by the paper. The schema combines flow statistics, protocol traits, contextual application metadata, and derived behavioural indicators.

Identifier-like control fields are not model predictors. Fields used only for grouping or window construction are marked with `model_input=false` in `feature_specification.csv`.

See `docs/feature_specification.csv` for the complete machine-readable table.
